# LetsEat

#### 项目介绍
LetsEat外卖点餐小程序，是以Spring Boot和微信特性为核心技术栈，实现一个从下单到接单流程完整，包含买家端和卖家端前后台功能的微信点餐系统。

#### 软件架构

开发环境：Windows 10 + IntelliJ IDEA 2018.2 + Maven 3.5.4+ JDK1.8

项目框架：Spring Boot + Spring-data-jpa

平台：Web / 微信测试号

数据库：MySQL 8.0
